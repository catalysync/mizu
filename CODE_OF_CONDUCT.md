# Code of Conduct

This project adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its code of conduct. All contributors, maintainers, and community members are expected to follow it.

## Scope

The code of conduct applies within all project spaces — issues, pull requests, discussions, commit messages, review comments, documentation — and when an individual is officially representing the project in public.

## Reporting

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported to the maintainers at **dami.func@gmail.com**.

All reports will be reviewed and investigated promptly and fairly. Maintainers are obligated to respect the privacy and security of the reporter of any incident.

## Enforcement

Maintainers will follow the [Community Impact Guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) in the Contributor Covenant when determining the consequences of any action they deem in violation of the code of conduct.
