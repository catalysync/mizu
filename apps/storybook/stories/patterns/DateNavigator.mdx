# DateNavigator

Preset chips + prev/next arrows + current-range label. The canonical "what time window am I looking at?" control for dashboards and reports.

## When to use

- Dashboard filter bar (charts, KPIs, tables scoped to a time range)
- Any feed / list / report where the user picks a time window

## Built-in presets

`today`, `yesterday`, `last-7`, `last-30`, `this-week`, `this-month`, `this-quarter`, `year-to-date`. Override via the `presets` prop to customize labels or add domain-specific ranges ("Fiscal Q1", "Since last sync").

## Accessibility

- Outer wrapper has `role="group"` with `aria-label="Date range"`
- Preset chips render as a `radiogroup` — keyboard-nav by tab + Space/Enter; `aria-checked` reflects the active preset
- Prev/Next arrows are labelled buttons (`aria-label="Previous range"` / `"Next range"`)
- The current-range label is `aria-live="polite"` so screen readers announce when the range shifts
