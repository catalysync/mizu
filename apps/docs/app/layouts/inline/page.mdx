# Inline

A layout primitive from the `@aspect/react` package.

## Import

```tsx
import { Inline } from '@aspect/react';
```

## CSS class

```css
.mizu-inline {
  /* ... */
}
```

## Usage

```tsx
<Inline gap="1rem">
  <div>Child 1</div>
  <div>Child 2</div>
</Inline>
```

## Props

| Prop        | Type     | Default       | Description        |
| ----------- | -------- | ------------- | ------------------ |
| `gap`       | `string` | Token default | CSS gap value      |
| `className` | `string` | —             | Additional classes |
