# Center

A layout primitive from the `@aspect/react` package.

## Import

```tsx
import { Center } from '@aspect/react';
```

## CSS class

```css
.mizu-center {
  /* ... */
}
```

## Usage

```tsx
<Center gap="1rem">
  <div>Child 1</div>
  <div>Child 2</div>
</Center>
```

## Props

| Prop        | Type     | Default       | Description        |
| ----------- | -------- | ------------- | ------------------ |
| `gap`       | `string` | Token default | CSS gap value      |
| `className` | `string` | —             | Additional classes |
