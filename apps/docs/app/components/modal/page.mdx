# Modal

A centered overlay dialog built on Radix Dialog. Includes backdrop, close button, and focus trap.

## Import

```tsx
import {
  Modal,
  ModalTrigger,
  ModalContent,
  ModalHeader,
  ModalTitle,
  ModalDescription,
  ModalBody,
  ModalFooter,
  ModalClose,
} from '@aspect/react';
```

## CSS class

```css
.mizu-dialog__content {
  /* ... */
}
```

## Props

Compound component pattern. Wrap in `<Modal>`, trigger with `<ModalTrigger>`, content in `<ModalContent>`.

## Accessibility

- Semantic HTML with proper ARIA roles
- Keyboard accessible
- Tested with vitest-axe
