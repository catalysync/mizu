# DropdownMenu

A contextual action menu. Built on Radix DropdownMenu.

## Import

```tsx
import {
  DropdownMenu,
  DropdownMenuTrigger,
  DropdownMenuContent,
  DropdownMenuItem,
  DropdownMenuLabel,
  DropdownMenuSeparator,
} from '@aspect/react';
```

## CSS class

```css
.mizu-dropdown-menu {
  /* ... */
}
```

## Props

| Prop (DropdownMenuItem) | Type      | Default | Description                          |
| ----------------------- | --------- | ------- | ------------------------------------ |
| `danger`                | `boolean` | `false` | Styles the item as destructive (red) |

## Accessibility

- Semantic HTML with proper ARIA roles
- Keyboard accessible
- Tested with vitest-axe
