# Textarea

A multi-line text input.

## Import

```tsx
import { Textarea } from '@aspect/react';
```

## CSS class

```css
.mizu-textarea {
  /* ... */
}
```

## Props

Accepts all native `<textarea>` attributes. Styled with the same focus ring and border tokens as Input.

## Accessibility

- Semantic HTML with proper ARIA roles
- Keyboard accessible
- Tested with vitest-axe
