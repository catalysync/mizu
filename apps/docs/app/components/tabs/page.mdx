# Tabs

Tabbed content navigation built on Radix Tabs.

## Import

```tsx
import { Tabs, TabsList, TabsTrigger, TabsContent } from '@aspect/react';
```

## CSS class

```css
.mizu-tabs {
  /* ... */
}
```

For non-React usage:

```html
<button class="mizu-tabs mizu-tabs--primary mizu-tabs--md">Label</button>
```

## Props

Compound component pattern. Wrap in `<Tabs defaultValue="...">`, use `<TabsList>` + `<TabsTrigger>` + `<TabsContent>`.

## Accessibility

- Uses semantic HTML elements with proper ARIA roles
- Keyboard navigable (Enter, Space where applicable)
- Tested with vitest-axe for WCAG compliance
- See the [Storybook](https://mizu-storybook.vercel.app) for interactive examples
