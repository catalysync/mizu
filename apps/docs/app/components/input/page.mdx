# Input

A text input field with size variants.

## Import

```tsx
import { Input } from '@aspect/react';
```

## CSS class

```css
.mizu-input {
  /* ... */
}
```

## Props

| Prop       | Type                   | Default | Description        |
| ---------- | ---------------------- | ------- | ------------------ |
| `size`     | `'sm' \| 'md' \| 'lg'` | `'md'`  | Input height       |
| `disabled` | `boolean`              | `false` | Disables the input |

Accepts all native `<input>` attributes.

## Accessibility

- Semantic HTML with proper ARIA roles
- Keyboard accessible
- Tested with vitest-axe
