# Input

A text input field. Supports sizes, labels, error states, and help text.

## Import

```tsx
import { Input } from '@aspect/react';
```

## Sizes

```tsx
<Input size="sm" placeholder="Small" />
<Input size="md" placeholder="Medium" />
<Input size="lg" placeholder="Large" />
```

## With label and help text

```tsx
<Input label="Email" helpText="We won't share this." />
```

## Error state

```tsx
<Input label="Price" error="Must be greater than 0" />
```

## Props

| Prop       | Type                   | Default | Description                  |
| ---------- | ---------------------- | ------- | ---------------------------- |
| `size`     | `'sm' \| 'md' \| 'lg'` | `'md'`  | Input height                 |
| `label`    | `string`               | —       | Auto-wraps in a form group   |
| `error`    | `string \| boolean`    | —       | Error message or error state |
| `helpText` | `ReactNode`            | —       | Help text below input        |
| `disabled` | `boolean`              | `false` | Disables the input           |

All native `<input>` attributes are also accepted.

## Guidelines

**Do:**

- Always provide a visible label or `aria-label`
- Show validation errors inline, not in toasts

**Don't:**

- Don't use placeholder text as a label substitute
