# Popover

An anchored overlay for inline editing or contextual content. Built on Radix Popover.

## Import

```tsx
import { Popover, PopoverTrigger, PopoverContent } from '@aspect/react';
```

## CSS class

```css
.mizu-popover {
  /* ... */
}
```

## Props

Use `showArrow` on `<PopoverContent>` to display a connecting arrow.

## Accessibility

- Semantic HTML with proper ARIA roles
- Keyboard accessible
- Tested with vitest-axe
