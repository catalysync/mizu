# Select

A dropdown select built on Radix Select. Supports grouped items, search, and custom rendering.

## Import

```tsx
import { Select, SelectTrigger, SelectContent, SelectItem, SelectValue } from '@aspect/react';
```

## CSS class

```css
.mizu-select__trigger {
  /* ... */
}
```

## Props

Compound component pattern. Wrap in `<Select>` root, use `<SelectTrigger>` + `<SelectContent>` + `<SelectItem>`.

## Accessibility

- Semantic HTML with proper ARIA roles
- Keyboard accessible
- Tested with vitest-axe
