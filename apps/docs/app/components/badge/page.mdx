# Badge

A small status indicator. Supports tones for semantic meaning and an optional dot indicator.

## Import

```tsx
import { Badge } from '@aspect/react';
```

## CSS class

```css
.mizu-badge {
  /* ... */
}
```

## Props

| Prop   | Type                                                        | Default     | Description                          |
| ------ | ----------------------------------------------------------- | ----------- | ------------------------------------ |
| `tone` | `'neutral' \| 'success' \| 'warning' \| 'danger' \| 'info'` | `'neutral'` | Color intent                         |
| `dot`  | `boolean`                                                   | `false`     | Shows a colored dot before the label |

## Accessibility

- Semantic HTML with proper ARIA roles
- Keyboard accessible
- Tested with vitest-axe
