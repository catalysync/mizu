# Card

A container with optional header, body, and footer sections. Use `interactive` for clickable cards.

## Import

```tsx
import { Card, CardHeader, CardBody, CardFooter } from '@aspect/react';
```

## CSS class

```css
.mizu-card {
  /* ... */
}
```

## Props

| Prop          | Type      | Default | Description                           |
| ------------- | --------- | ------- | ------------------------------------- |
| `interactive` | `boolean` | `false` | Adds hover effect for clickable cards |

Sub-components: `CardHeader` (title, description), `CardBody`, `CardFooter`.

## Accessibility

- Semantic HTML with proper ARIA roles
- Keyboard accessible
- Tested with vitest-axe
