# EmptyState

A placeholder for screens with no data.

<Preview>
  <EmptyState
    title="No results"
    description="Try adjusting your search."
    actions={<Button variant="primary">Clear filters</Button>}
  />
</Preview>

## Import

```tsx
import { EmptyState } from '@aspect/react';
```

## With icon

```tsx
<EmptyState
  icon={<svg>...</svg>}
  title="Inbox empty"
  description="All conversations are resolved."
/>
```

## Props

| Prop          | Type        | Default  | Description     |
| ------------- | ----------- | -------- | --------------- |
| `icon`        | `ReactNode` | —        | Decorative icon |
| `title`       | `ReactNode` | required | Heading         |
| `description` | `ReactNode` | —        | Supporting text |
| `actions`     | `ReactNode` | —        | CTA buttons     |

## Guidelines

**Do:**

- Always include a title
- Suggest an action when possible

**Don't:**

- Don't use for loading states — use a skeleton or spinner
