# Tooltip

A small popup on hover. Built on Radix Tooltip.

## Import

```tsx
import { TooltipProvider, Tooltip, TooltipTrigger, TooltipContent } from '@aspect/react';
```

## Usage

Wrap your app in `TooltipProvider` once. Then use Tooltip anywhere.

```tsx
<TooltipProvider>
  <Tooltip>
    <TooltipTrigger asChild>
      <Button>Hover me</Button>
    </TooltipTrigger>
    <TooltipContent>Helpful text</TooltipContent>
  </Tooltip>
</TooltipProvider>
```

## Props (TooltipContent)

| Prop         | Type                                     | Default | Description                |
| ------------ | ---------------------------------------- | ------- | -------------------------- |
| `showArrow`  | `boolean`                                | `true`  | Display a connecting arrow |
| `side`       | `'top' \| 'right' \| 'bottom' \| 'left'` | `'top'` | Placement                  |
| `sideOffset` | `number`                                 | `6`     | Distance from trigger      |
