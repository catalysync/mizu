# Button

Triggers an action. Supports variants, sizes, loading state, and polymorphic rendering via `asChild`.

## Import

```tsx
import { Button } from '@aspect/react';
```

## CSS class

```css
.mizu-button {
  /* ... */
}
```

## Props

| Prop       | Type                                                   | Default     | Description                                        |
| ---------- | ------------------------------------------------------ | ----------- | -------------------------------------------------- |
| `variant`  | `'primary' \| 'secondary' \| 'ghost' \| 'destructive'` | `'primary'` | Visual style                                       |
| `size`     | `'sm' \| 'md' \| 'lg' \| 'icon'`                       | `'md'`      | Size preset                                        |
| `loading`  | `boolean`                                              | `false`     | Shows spinner, disables interaction                |
| `disabled` | `boolean`                                              | `false`     | Disables the button                                |
| `asChild`  | `boolean`                                              | `false`     | Render as child element (e.g. Link) via Radix Slot |
| `type`     | `'button' \| 'submit' \| 'reset'`                      | `'button'`  | HTML button type                                   |

## Accessibility

- Semantic HTML with proper ARIA roles
- Keyboard accessible
- Tested with vitest-axe
