# Separator

A visual divider. Built on Radix Separator.

## Import

```tsx
import { Separator } from '@aspect/react';
```

## Usage

```tsx
<Separator />
<Separator orientation="vertical" />
```

## Props

| Prop          | Type                         | Default        | Description |
| ------------- | ---------------------------- | -------------- | ----------- |
| `orientation` | `'horizontal' \| 'vertical'` | `'horizontal'` | Direction   |
