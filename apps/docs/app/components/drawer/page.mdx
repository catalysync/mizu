# Drawer

A side panel overlay built on Radix Dialog. Slides in from any edge.

## Import

```tsx
import {
  Drawer,
  DrawerTrigger,
  DrawerContent,
  DrawerHeader,
  DrawerTitle,
  DrawerBody,
  DrawerFooter,
} from '@aspect/react';
```

## CSS class

```css
.mizu-drawer__content {
  /* ... */
}
```

## Props

| Prop (DrawerContent) | Type                                     | Default   | Description     |
| -------------------- | ---------------------------------------- | --------- | --------------- |
| `side`               | `'left' \| 'right' \| 'top' \| 'bottom'` | `'right'` | Slide direction |

## Accessibility

- Semantic HTML with proper ARIA roles
- Keyboard accessible
- Tested with vitest-axe
